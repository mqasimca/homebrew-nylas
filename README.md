# Homebrew Tap for Nylas CLI

Official Homebrew tap for [Nylas CLI](https://github.com/mqasimca/nylas) - a command-line interface for the Nylas API.

## Installation

```bash
brew install mqasimca/nylas/nylas
```

Or manually tap first:

```bash
brew tap mqasimca/nylas
brew install nylas
```

## Updating

```bash
brew upgrade nylas
```

## Uninstalling

```bash
brew uninstall nylas
brew untap mqasimca/nylas  # optional
```

## About Nylas CLI

Nylas CLI lets you manage email, calendar, and contacts from the command line using the Nylas v3 API.

For more information, see the [main repository](https://github.com/mqasimca/nylas).
